# CodeAlpha Student Grade Tracker (Task 1)

##  Project Description
This is a simple Java console-based application developed as "Task 1" of the **CodeAlpha Java Programming Internship**.

The program allows the user to:
- Enter student names and marks
- Calculate average, highest, and lowest marks
- Display a clear student grade summary

The project demonstrates basic Java concepts such as loops, arrays, input handling, and conditional statements.


##  Technologies Used
- Java
- VS Code
- Git & GitHub

---

##  How to Run the Project

1. Open the project folder in **VS Code**
2. Open terminal inside the folder
3. Compile the program:
   ```bash
   javac StudentGradeTracker.java
